# wikidata

SPARQL queries and Python scripts for visualizing Wikidata on Black artists in Philadelphia. Part of the [ArtVizPhilly](https://artvizphilly.github.io/) project.

## Contents

- `notebooks/` -- Jupyter notebooks for querying and visualizing Wikidata
- `scripts/` -- Standalone Python script version
- `data/` -- Archived query results (2020, 2021)

## Setup

```bash
pip install -r requirements.txt
jupyter notebook notebooks/Querying_and_Visualizing_Wikidata.ipynb
```

Or run without Jupyter:

```bash
python scripts/querying_and_visualizing_wikidata.py
```
