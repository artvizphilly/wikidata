# wikidata

SPARQL queries and Python scripts for visualizing Wikidata on Black artists in Philadelphia. Part of the [ArtVizPhilly](https://artvizphilly.github.io/) project.

## Contents

- `Querying_and_Visualizing_Wikidata.ipynb` -- Main notebook: SPARQL queries, data wrangling, Plotly visualizations
- `Query and Visualize Wikidata.ipynb` -- Alternate analysis notebook
- `Wikidata_SPARQL_Queries.ipynb` -- SPARQL syntax reference
- `querying_and_visualizing_wikidata.py` -- Standalone script version of the main notebook
- `Wikidata Query *.csv` -- Archived query results (2020, 2021)

## Setup

```bash
pip install -r requirements.txt
jupyter notebook Querying_and_Visualizing_Wikidata.ipynb
```

Or run without Jupyter:

```bash
python querying_and_visualizing_wikidata.py
```
